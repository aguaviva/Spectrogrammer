# Spectrogrammer
Android app that shows a spectrogram

## Features
 - Low latency audio (uses openSL and native code)
 - Up to 8192 samples FFT (using FFTW)
 - Log/Linear scales
 - Configurable FFT overlap
 - Vertical bars smoothing 
 - Measuring cursor
